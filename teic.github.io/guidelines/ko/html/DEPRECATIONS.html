TEI: About the Deprecation Appendix¶부록 G.1 About the Deprecation Appendix

This appendix gives you a list of all items (elements, attributes, attribute values, datatypes etc.) which are deprecated in this revision of the TEI Guidelines (revision 3c0c64e of TEI P5 Version 3.5.0).

IdentifierComponent typeThis feature will be withdrawn afterDescription
teidata.point / deprecate_trailing_decimal_pointconstraintSpec2019-02-25Several standards bodies, including NIST in the USA, strongly recommend against ending the representation of a number with a decimal point. So instead of ‘3.’ use either ‘3’ or ‘3.0’.
content / empty-content-deprecatedconstraintSpec2019-08-25Use a child <empty> element to indicate that the element being specified is not allowed to have content.
