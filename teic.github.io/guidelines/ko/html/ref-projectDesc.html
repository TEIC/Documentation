<projectDesc>

<projectDesc> (프로젝트 기술) 전자 파일이 부호화된 목적을 상세히 기술하며, 아울러 그것이 수집된 절차에 관한 기타 관련 정보를 기술한다, . [2.3.1 The Project Description 2.3 The Encoding Description 15.3.2 Declarable Elements]
모듈header — The TEI Header
속성att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.declarable (@default)
클럽 회원
model.encodingDescPart
에 의해 포함된
header: encodingDesc
포함할 수 있다
core: p
linking: ab
예
<projectDesc>
 <p>Texts collected for use in the Claremont Shakespeare Clinic, June 1990</p>
</projectDesc>
