att.personal

att.personal (사람 이름의 성분에 대한 속성) 사람 이름의 부분을 형성하는 요소에 대한 공통 속성 [13.2.1 Personal Names]
모듈tei — The TEI Infrastructure
원소addName forename genName name objectName orgName persName placeName roleName surname
속성att.naming (@role, @nymRef) (att.canonical (@key, @ref))
full full¶축약 또는 이니셜로 간단하게 이름 성분이 완전히 제시된 것인지의 여부를 나타낸다.
상태 수의적
자료 유형 teidata.enumerated
적법한 값은:
yes
이름 성분이 완전히 제시된다. [기본값]
abb
(축약된) 이름 성분이 축약형으로 제시된다.
init
(이니셜 문자) 이름 성분이 하나의 이니셜(첫글자)로 제시되었다.
sort sort¶사람 이름 내에 다른 것과 관련된 이름 성분의 정렬 순서를 명시한다.
상태 수의적
자료 유형 teidata.count
