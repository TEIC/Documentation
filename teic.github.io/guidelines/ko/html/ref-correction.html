<correction>

<correction> (수정 원리) 텍스트의 수정의 상황과 방법에 대해 설명한다. [2.3.3 The Editorial Practices Declaration 15.3.2 Declarable Elements]
모듈header — The TEI Header
속성att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.declarable (@default)
status status¶텍스트에 적용된 수정의 정도를 표시한다.
상태 수의적
자료 유형 teidata.enumerated
적법한 값은:
high
텍스트가 철저하게 검사되고 교정되었다.
medium
텍스트가 적어도 한번 검사되었다.
low
텍스트가 검사되지 않았다.
unknown
텍스트의 교정 상태를 알 수 없다.
method method¶텍스트의 수정사항을 표시하기 위해 채택한 방법을 나타낸다.
상태 수의적
자료 유형 teidata.enumerated
적법한 값은:
silent
수정사항이 표시되지 않았다. [기본값]
markup
수정사항이 마크업을 사용해 표사되었다.
클럽 회원
model.editorialDeclPart
에 의해 포함된
header: editorialDecl
포함할 수 있다
core: p
linking: ab
주석

May be used to note the results of proof reading the text against its original, indicating (for example) whether discrepancies have been silently rectified, or recorded using the editorial tags described in section 3.4 Simple Editorial Changes.

예
<correction>
 <p>Errors in transcription controlled by using the WordPerfect spelling checker, with a user
   defined dictionary of 500 extra words taken from Chambers Twentieth Century
   Dictionary.</p>
</correction>
