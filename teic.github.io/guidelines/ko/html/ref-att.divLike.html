att.divLike

att.divLike 구역과 동일한 방식으로 처리되는 모든 요소에 공통적인 속성을 제공한다. [4 Default Text Structure]
모듈tei — The TEI Infrastructure
원소div div1 div2 div3 div4 div5 div6 div7 lg
속성att.metrical (@met, @real, @rhyme) att.fragmentable (@part)
org org¶(조직) 구역 내용이 조직된 방법에 대해 명시한다.
상태 수의적
자료 유형 teidata.enumerated
적법한 값은:
composite
혼합 내용: 즉, 이 구역의 직접적 내용이 처리되는 순서 또는 상호 관련성에 관해 어떠한 주장도 없다.
uniform
통일 내용: 즉, 이 요소의 직접적 내용이 차례대로 처리되는 논리적 단위를 형성하는 것으로 간주된다. [기본값]
sample sample¶이 구역이 원본의 표본이며 어떤 부분인지를 나타낸다.
상태 수의적
자료 유형 teidata.enumerated
적법한 값은:
initial
구역이 원본의 후반부에서 제시된 자료가 부족하다.
medial
구역이 초반부와 후반부 자료가 부족하다.
final
구역이 초반부 자료가 부족하다.
unknown
표본 자료의 원본 내의 위치를 알 수 없다.
complete
구역이 표본이 아니다. [기본값]
