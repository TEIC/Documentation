<fsdLink>

<fsdLink> (자질 구조 선언 연결) 자질 구조 선언에 유형화된 자질 구조의 이름을 관련시킨다. [18.11 Feature System Declaration]
모듈iso-fs — Feature Structures
속성att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
type type¶기록된 자질 구조의 유형을 식별한다; 이것은 적어도 하나 이상의 자질 구조에 대한 유형 값이 될 것이다.
상태 필수적
자료 유형 teidata.enumerated
target target¶현 문서 내 또는 다른 문서에서 자질 구조 선언 (fsDecl) 요소에 대한 포인터를 제공한다.
상태 필수적
자료 유형 teidata.pointer
에 의해 포함된
iso-fs: fsdDecl
포함할 수 있다공백 요소
예
<fsdLink type="subentry"
 target="http://www.example.com/fsdLib.xml#L1234"/>
