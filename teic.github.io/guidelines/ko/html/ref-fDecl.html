<fDecl>

<fDecl> (자질 선언) 단일 자질, 자질 이름 명시, 조직, 허용 값의 범위, 그리고 수의적으로 기본 값을 선언한다. [18.11 Feature System Declaration]
모듈iso-fs — Feature Structures
속성att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
name name¶선언되고 있는 자질 이름을 제시한다; 텍스트 내 f 요소의 name 속성과 일치한다.
상태 필수적
자료 유형 teidata.name
optional optional¶이 자질 값의 제시 여부를 표시한다.
상태 수의적
자료 유형 teidata.truthValue
기본값 true
에 의해 포함된
iso-fs: fsDecl
포함할 수 있다
iso-fs: fDescr vDefault vRange
예
<fDecl name="INV">
 <fDescr>inverted sentence</fDescr>
 <vRange>
  <vAlt>
   <binary value="true"/>
   <binary value="false"/>
  </vAlt>
 </vRange>
 <vDefault>
  <binary value="false"/>
 </vDefault>
</fDecl>

   att.global.rendition.attributes,
