예

The measure element may be used to supply normalised or machine tractable versions of the size or sizes concerned.

<extent>
 <measure unit="MiB" quantity="4.2">About four megabytes</measure>
 <measure unit="pages" quantity="245">245 pages of source
   material</measure>
</extent>
