<setting>

<setting> 언어 상호작용이 발생하는 특정 무대를 기술한다. [15.2.3 The Setting Description]
모듈corpus — Language Corpora
속성att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.ascribed (@who)
에 의해 포함된
corpus: settingDesc
포함할 수 있다
core: date name p time
corpus: activity locale
linking: ab
namesdates: orgName persName
주석

If the who attribute is not supplied, the setting is assumed to be that of all participants in the language interaction.

예
<setting>
 <name>New York City, US</name>
 <date>1989</date>
 <locale>on a park bench</locale>
 <activity>feeding birds</activity>
</setting>

   att.global.linking.attributes,
