att.entryLike raggruppa differenti stili di voci di dizionario. [9.1 Dictionary Body and Overall Structure 9.2 The Structure of Dictionary Entries]

AttributiAttributiatt.typed (type, @subtype)
type type¶indica il tipo di voce, nei dizionari con tipi multipli.
Stato Opzionale
Tipo di dati teidata.enumerated
I valori suggeriti includono:
main
una voce principale (valore predefinito). [Valore di default]
hom
(omografo) un omografo con una voce separata.
xref
(riferimento incrociato) una voce ridotta la cui sola funzione è di indirizzare ad una voce principale (ad esempio per le forme di un verbo irregolare o per varianti ortografiche: was che indirizza a be, o esthete per aesthete).
affix
una voce per un prefisso, infisso o suffisso.
abbr
(abbreviazione) una voce per un'abbreviazione.
supplemental
una voce supplementare (in uso nei dizionari che pubblicano supplementi alle opere principali in cui includono informazioni aggiornate sulle voci).)
foreign
una voce per una parola starniera in un dizionario monolingue.
