<fDecl>

<fDecl> (dichiarazione di tratti) dichiara un singolo tratto, specificandone il nome, l'organizzazione, la gamma di valori permessi, e opzionale il suo valore predefinito. [18.11 Feature System Declaration]
Moduloiso-fs — Feature Structures
Attributiatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
name name¶indica il nome del tratto che si sta dichiarando; corrisponde all'attributo name dell'elemento f nel testo.
Stato Richiesto
Tipo di dati teidata.name
optional optional¶indica se il valore del tratto possa o meno essere presente
Stato Opzionale
Tipo di dati teidata.truthValue
Valore di default true
Contenuto in
iso-fs: fsDecl
Può contenere
iso-fs: fDescr vDefault vRange
Esempio
<fDecl name="INV">
 <fDescr>inverted sentence</fDescr>
 <vRange>
  <vAlt>
   <binary value="true"/>
   <binary value="false"/>
  </vAlt>
 </vRange>
 <vDefault>
  <binary value="false"/>
 </vDefault>
</fDecl>
