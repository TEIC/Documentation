<unicodeName>

<unicodeName> (nome della proprietà Unicode) contiene il nome di una proprietà Unicode registrata normativa o informativa. [5.2.1 Character Properties]
Modulogaiji — Characters, Glyphs, and Writing Modes
Attributiatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
version version¶specifica il numero della versione dello standard Unicode nel quale è definito il nome della proprietà.
Stato Opzionale
Tipo di dati teidata.version
Contenuto in
gaiji: charProp
Può contenereAmmessi soltanto caratteri di testo
Nota

A definitive list of current Unicode property names is provided in The Unicode Standard.

Esempio
<unicodeName>character-decomposition-mapping</unicodeName>
<unicodeName>general-category</unicodeName>
