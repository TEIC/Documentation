macro.specialPara

macro.specialPara (contenuto di paragrafo "speciale") definisce il modello di contenuto di elementi quali note o voci di lista che contiene una serie di elementi a livello di componenti aventi la stessa struttura di un paragrafo con una serie di elementi a livello sintagmatico e interlivello [1.3 The TEI Class System]
