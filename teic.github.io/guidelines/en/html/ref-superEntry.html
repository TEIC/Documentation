<superEntry>

<superEntry> groups a sequence of entries within any kind of lexical resource, such as a dictionary or lexicon which function as a single unit, for example a set of homographs. [9.1 Dictionary Body and Overall Structure]

Example
<superEntry>
 <form>
  <orth>abandon</orth>
  <hyph>a|ban|don</hyph>
  <pron>@"band@n</pron>
 </form>
 <entry n="1">
  <gramGrp>
   <pos>v</pos>
   <subc>T1</subc>
  </gramGrp>
  <sense n="1">
   <def>to leave completely and for ever ... </def>
  </sense>
  <sense n="2"/>
 </entry>
 <entry n="2">
  <gramGrp>
   <pos>n</pos>
   <subc>U</subc>
  </gramGrp>
  <def>the state when one's feelings and actions are uncontrolled; freedom from
     control</def>
 </entry>
</superEntry>
