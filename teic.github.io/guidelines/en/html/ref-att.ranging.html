att.ranging

att.ranging provides attributes for describing numerical ranges.
Moduletei — The TEI Infrastructure
Membersatt.dimensions [att.damaged [damage damageSpan] add addSpan age birth date death del delSpan depth dim dimensions ex floruit gap geogFeat height mod offset origDate population redo restore retrace secl space state subst substJoin supplied surplus time trait unclear undo width] num precision
AttributesAttributes
atLeast atLeast¶gives a minimum estimated value for the approximate measurement.
Status Optional
Datatype teidata.numeric
atMost atMost¶gives a maximum estimated value for the approximate measurement.
Status Optional
Datatype teidata.numeric
min min¶where the measurement summarizes more than one observation or a range, supplies the minimum value observed.
Status Optional
Datatype teidata.numeric
max max¶where the measurement summarizes more than one observation or a range, supplies the maximum value observed.
Status Optional
Datatype teidata.numeric
confidence confidence¶specifies the degree of statistical confidence (between zero and one) that a value falls within the range specified by min and max, or the proportion of observed values that fall within that range.
Status Optional
Datatype teidata.probability
