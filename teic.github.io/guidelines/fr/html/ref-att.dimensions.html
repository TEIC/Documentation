att.dimensions

att.dimensions fournit des attributs pour décrire la taille des objets physiques

Membresatt.damaged [damage damageSpan] add addSpan age birth date death del delSpan depth dim dimensions ex floruit gap geogFeat height mod offset origDate population redo restore retrace secl space state subst substJoin supplied surplus time trait unclear undo width
Attributsatt.ranging (@atLeast, @atMost, @min, @max, @confidence)
unit unit¶noms des unités utilisées pour la mesure.
Statut Optionel
Type de données teidata.enumerated
Les valeurs suggérées comprennent:
cm
(centimètres)
mm
(millimètres)
in
(pouces)
lines
lignes de texte
chars
(characters) caractères du texte
quantity quantity¶spécifie la longueur dans les unités indiquées
Statut Optionel
Type de données teidata.numeric
extent extent¶indique la dimension de l'objet en utilisant un vocabulaire spécifique à un projet qui combine la quantité et l'unité dans une chaîne seule de mots.

scope scope¶spécifie l'applicabilité de cette mesure, là où plus d'un objet est mesuré.
Statut Optionel
Type de données teidata.enumerated
Exemple de valeurs possibles:
all
la mesure s'applique à tous les cas.
most
la mesure s'applique à la plupart des cas examinés
range
la mesure s'applique seulement à l'ensemble des exemples indiqués.
