<revisionDesc>

<revisionDesc> (descriptif des révisions) fournit un résumé de l’historique des révisions d’un fichier. [2.6 The Revision Description 2.1.1 The TEI Header and Its Components]
Moduleheader — The TEI Header
Attributsatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.docStatus (@status)
Contenu dans
header: teiHeader
Peut contenir
core: list
header: change listChange
Note

Les changements les plus récents apparaissent en début de liste

Exemple
<revisionDesc>
 <list>
  <item>
   <date when="2003-04-12">12 avril 03</date>Dernière révision par F. B.</item>
  <item>
   <date when="2003-03-01">1 mars 03</date> F.B a fait le nouveau fichier.</item>
 </list>
</revisionDesc>
Toute la liste
