<seriesStmt>

<seriesStmt> (mention de collection) regroupe toute information relative à la collection (si elle existe) à laquelle appartient une publication. [2.2.5 The Series Statement 2.2 The File Description]
Moduleheader — The TEI Header
Attributsatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
Contenu dans
header: biblFull fileDesc
Peut contenir
core: biblScope editor p respStmt title
header: idno
linking: ab
Exemple
<seriesStmt>
 <title>Babel</title>
 <respStmt>
  <resp>directeur de collection</resp>
  <name>Jacques Dubois</name>
 </respStmt>
 <respStmt>
  <resp>directeur de collection</resp>
  <name>Hubert Nyssen</name>
 </respStmt>
 <idno type="ISSN">1140-3853</idno>
</seriesStmt>
