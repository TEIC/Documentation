att.rdgPart attributs d'éléments qui marquent le début ou la fin d'un manuscrit fragmentaire ou d'un autre témoin. [12.1.5 Fragmentary Witnesses]

wit wit¶(témoin ou témoins) contient une liste d'une ou plusieurs abréviations désignant les témoins qui commencent ou finissent à ce point.
Statut Optionel
Type de données 1–∞ occurrences de teidata.pointer séparé par un espace
Note
