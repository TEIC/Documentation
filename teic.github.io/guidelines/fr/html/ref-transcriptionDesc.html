<transcriptionDesc>

<transcriptionDesc> describes the set of transcription conventions used, particularly for spoken material. [8.2 Documenting the Source of Transcribed Speech]
Modulespoken — Transcriptions of Speech
Attributsatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
ident ident¶supplies an identifier for the encoding convention, independent of any version number.
Statut Requis
Type de données teidata.name
version version¶supplies a version number for the encoding conventions used, if any.
Statut Optionel
Type de données teidata.versionNumber

 version="2004"/>
