<eLeaf>

<eLeaf> (feuille ou nœud terminal d'un arbre englobant) la feuille d'un arbre englobant, qui peut aussi être encodée au moyen d'un élément eTree [19.3 Another Tree Notation]
Modulenets — Graphs, Networks, and Trees
Attributsatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.typed (@type, @subtype)
value value¶fournit l'identifiant de l'objet représenté par la feuille de l'arbre englobant ; cet objet peut être une paire nom-valeur ou tout autre élément analytique.
Statut Optionel
Type de données teidata.pointer
Contenu dans
nets: eTree triangle
Peut contenir
core: label ptr ref
Note

La balise eTree peut être utilisée si l'encodeur ne souhaite pas faire une distinction de nom entre des nœuds terminaux (feuilles) ou non-terminaux dans des arbres englobants ; ils sont distingués par leur disposition.

Exemple
<eLeaf value="http://an.fsurl.tei/#FSWITH">
 <label>avec</label>
</eLeaf>
