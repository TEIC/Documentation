macro.phraseSeq

macro.phraseSeq (suite de syntagmes.) définit un ordre de données et d'éléments syntagmatiques. [1.4.1 Standard Content Models]

 <alternate minOccurs="0"

Declaration
<rng:define name="macro.phraseSeq">
 <rng:zeroOrMore>
  <rng:choice>
   <rng:text/>
   <rng:ref name="model.gLike"/>
   <rng:ref name="model.qLike"/>
   <rng:ref name="model.phrase"/>
   <rng:ref name="model.global"/>
  </rng:choice>
 </rng:zeroOrMore>
</rng:define>
macro.phraseSeq =
   ( text | model.gLike | model.qLike | model.phrase | model.global )*
