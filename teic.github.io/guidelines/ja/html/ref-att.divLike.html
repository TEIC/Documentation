att.divLike

att.divLike 区分(div, division)に相当する全要素に共通の属性を示す． [4 Default Text Structure]
モジュールtei — The TEI Infrastructure
構成div div1 div2 div3 div4 div5 div6 div7 lg
属性att.metrical (@met, @real, @rhyme) att.fragmentable (@part)
org org¶ (organization) 当該区分の内容がどのように構成されているかを示す．
状態 任意
データ型 teidata.enumerated
正当な値:
composite
合成内容．当該区分の内容が処理される順番，すなわち内部関 係には制約がない．
uniform
統一内容．当該区分の内容が処理される際，ひとつの論理的単位を 構成していると見なす． [初期値]
sample sample¶当該区分が，元資料のものを含むかどうか，そうであればその場所はど こかを示す．
状態 任意
データ型 teidata.enumerated
正当な値:
initial
当該区分は，元資料の終わりの部分が欠けている．
medial
当該区分は，元資料の始めと終わりの部分が欠けている．
final
当該区分は，元資料の始めの部分が欠けている．
unknown
元資料のどの部分かは不明．
complete
当該領域は，元資料の一部ではない． [初期値]
