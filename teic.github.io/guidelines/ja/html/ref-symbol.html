<symbol>

<symbol> (symbolic value) 記号リスト中のいち項目を含む素性値規定のうち，値の部分を示す． [18.3 Other Atomic Feature Values]
モジュールiso-fs — Feature Structures
属性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.datcat (@datcat, @valueDatcat)
value value¶当該組成の記号値を示す．素性宣言のリスト中にあるいち項目になる．
状態 必須
