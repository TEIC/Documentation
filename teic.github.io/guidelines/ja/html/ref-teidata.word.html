teidata.word

teidata.word いち単語またはトークンをとる属性値の範囲を定義する．
モジュールtei — The TEI Infrastructure

Content model
<content>
 <dataRef name="token"
  restriction="(\p{L}|\p{N}|\p{P}|\p{S})+"/>
</content>
宣言
<rng:define name="teidata.word">
 <rng:data type="token">
  <rng:param name="pattern">(\p{L}|\p{N}|\p{P}|\p{S})+</rng:param>
 </rng:data>
</rng:define>
teidata.word = token { pattern = "(\p{L}|\p{N}|\p{P}|\p{S})+" }
解説

当該データ型を使用する属性値は，ひとつの‘単語’ になる．単語とは，文字，数字，句読点などの記号から構成されている． 空白文字は含むことができない．
