teidata.truthValue

teidata.truthValue 真偽値を示す属性値の範囲を定義する．

宣言
<rng:define name="teidata.truthValue">
 <rng:data type="boolean"/>
</rng:define>
teidata.truthValue = xsd:boolean
解説

曖昧な値が許されない場合に使われるデータ型になる．属性値が真偽値で ない場合，例えば"unknown"や"inapplicable"の場合には，当該データ型 の拡張版であるdata.xTruthValueを使うべきである．
