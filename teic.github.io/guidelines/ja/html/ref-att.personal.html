att.personal

att.personal  (attributes for components of names usually, but not necessarily, personal names) 個人名の構成要素となる要素に付与される属性． [13.2.1 Personal Names]
モジュールtei — The TEI Infrastructure
構成addName forename genName name objectName orgName persName placeName roleName surname
属性att.naming (@role, @nymRef) (att.canonical (@key, @ref))
full full¶当該名前要素は省略がないか，省略形か，イニシャルのような簡単なも のか示す．
状態 任意
データ型 teidata.enumerated
正当な値:
yes
当該名前要素は，省略無く示されている． [初期値]
abb
(abbreviated) 当該名前要素には，省略がある．
init
(initial letter) 当該名前要素は，イニシャルだけで示されている．
sort sort¶個人名間で並び替えをする際の順番を示す．
状態 任意
データ型 teidata.count
