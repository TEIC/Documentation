<sealDesc>

<sealDesc> (seal description) 手書き資料に付属するシールや他の付着物について，一連の段落や，一連の 要素sealにより，可能であれば要素decoNoteと共に示す． [10.7.3.2 Seals]
モジュールmsdescription — Manuscript Description
属性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
クラブのメンバー
model.physDescPart
上位
msdescription: physDesc
下位
core: p
linking: ab
msdescription: condition decoNote seal summary
例
<sealDesc>
 <seal type="pendant" contemporary="true">
  <p>Green wax vertical oval seal attached at base.</p>
 </seal>
</sealDesc>
一括表示 
例
<sealDesc>
 <p>Parchment strip for seal in place; seal missing.</p>
</sealDesc>
