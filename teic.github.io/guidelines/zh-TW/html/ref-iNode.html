<iNode>

<iNode> (中間 (或內部) 節點) 表示一個樹狀結構的中間 (或內部) 節點。 [19.2 Trees]
組件nets — Graphs, Networks, and Trees
屬性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
value value¶提供一個中間節點值，該節點為一個功能結構或其他分析元素。
狀態 非必備的
資料類型 teidata.pointer
children children¶提供中間節點的子節點之元素識別符碼列表。
狀態 必備的
資料類型 1–∞ 次可出現: teidata.pointer 以空白鍵隔開
parent parent¶提供中間節點的雙親節點的元素識別符碼。
狀態 非必備的
資料類型 teidata.pointer
ord ord¶ (ordered) 說明內部節點是否整齊排列。
狀態 非必備的
資料類型 teidata.xTruthValue
follow follow¶提供此節點所尾隨的元素識別符碼。
狀態 非必備的
資料類型 teidata.pointer
outDegree outDegree¶中間節點的向外分支度，即為其子節點的數量。
狀態 非必備的
資料類型 teidata.count
可包含在於
nets: tree
可包含
core: label
例子
<iNode xml:id="pt1" children="#GD-UP1"
 parent="#GD-VB1" follow="#GD-PN1" outDegree="1">
 <label>PT</label>
</iNode>
