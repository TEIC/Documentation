att.spanning

att.spanning 提供元素的屬性，這些元素使用參照機制來限定某一文字段，而非包含此文字段。 [11.3.1.4 Additions and Deletions 1.3.1 Attribute Classes]

屬性屬性
spanTo spanTo¶指出文字段的結尾，該文字段以帶有此屬性的元素開頭。
狀態 非必備的
資料類型 teidata.pointer
SchematronThe @spanTo attribute must point to an element following the current element
