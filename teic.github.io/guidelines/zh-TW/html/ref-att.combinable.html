att.combinable

att.combinable provides attributes indicating how multiple references to the same object in a schema should be combined
組件tagdocs — Documentation Elements
成員att.identified [attDef classSpec constraintSpec dataSpec elementSpec macroSpec moduleSpec paramSpec schemaSpec] valDesc valItem valList
屬性att.deprecated (@validUntil)
mode mode¶標明該宣告在其來源模組中的作用。
狀態 非必備的
資料類型 teidata.enumerated
合法的值是:
add
該宣告添加在現有定義中 [預設值]
delete
該宣告以及所有子宣告從現有的設定中被移除
change
該宣告改變在現有定義中名稱相同的宣告。
replace
該宣告代替在現有定義中名稱相同的宣告
