<valList>

<valList> (屬性值列表) 包含一個或多個元素valItem，以定義一個屬性可使用的屬性值。 [22.5.3 Attribute List Specification]
組件tagdocs — Documentation Elements
屬性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.combinable (@mode) (att.deprecated (@validUntil))
type type¶標明該屬性值列表的延展性。
狀態 非必備的
資料類型 teidata.enumerated
合法的值是:
closed
僅允許標明的屬性值。
semi
(semi-open) 所有標明的屬性值應該維持有效；但其他屬性值亦為合法，且所使用的軟體對於它們應該具有適當的後備處理程序。
open
所標明的屬性值僅為樣本屬性值。 [預設值]
