att.dimensions

att.dimensions 提供符合某種度量值的屬性值。
組件tei — The TEI Infrastructure
成員att.damaged [damage damageSpan] add addSpan age birth date death del delSpan depth dim dimensions ex floruit gap geogFeat height mod offset origDate population redo restore retrace secl space state subst substJoin supplied surplus time trait unclear undo width
屬性att.ranging (@atLeast, @atMost, @min, @max, @confidence)
unit unit¶度量單位的名稱。
狀態 非必備的
資料類型 teidata.enumerated
被推薦的值包含:
cm
(centimetres)
mm
(millimetres)
in
(inches)
lines
lines of text
chars
(characters) characters of text
quantity quantity¶specifies the length in the units specified
狀態 非必備的
資料類型 teidata.numeric
extent extent¶indicates the size of the object concerned using a project-specific vocabulary combining quantity and units in a single string of words.
狀態 非必備的
資料類型 teidata.text
<gap extent="5 words"/>
bibliography 
<height extent="half the page"/>
bibliography 
precision precision¶characterizes the precision of the values specified by the other attributes.
狀態 非必備的
資料類型 teidata.certainty
scope scope¶測量多個物件時，標明此度量的可應用範圍。
狀態 非必備的
資料類型 teidata.enumerated
實例值包含:
all
度量可應用於所有實例。
most
度量可應用於大部分實例。
range
度量僅應用於特定範圍的實例。
