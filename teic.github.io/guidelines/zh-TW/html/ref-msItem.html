<msItem>

<msItem> 描述手稿或手稿部分的智慧內容中所包含的單一作品或項目。 [10.6.1 The msItem and msItemStruct Elements]

例子
<msItem>
 <locus>f.495</locus>
 <title>台灣隨筆</title>
 <incipit>幾荷蘭人由洋中來，假地日本，久而不帰，遂築城而有之。</incipit>
 <explicit>惟商舶可以航海，凡使節往來咸藉之。</explicit>
 <textLang mainLang="zh-tw">繁體中文</textLang>
</msItem>
