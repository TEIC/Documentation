macro.specialPara

macro.specialPara (「特殊」段落內容) 定義元素的內容模型，如註解或列表項目，這些內容模型包含一系列組合性層次元素，或與一個段落有相同的結構，包含一系列的詞組層次或inter-層次元素。 [1.3 The TEI Class System]
