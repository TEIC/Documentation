att.divLike

att.divLike 提供一組屬性，通用於所有與區段作用相 同的元素。 [4 Default Text Structure]
組件tei — The TEI Infrastructure
成員div div1 div2 div3 div4 div5 div6 div7 lg
屬性att.metrical (@met, @real, @rhyme) att.fragmentable (@part)
org org¶ (organization) 說明該區段內容的組合方式。
狀態 非必備的
資料類型 teidata.enumerated
合法的值是:
composite
複合內容：未說明此區段目前內容的處理順序或相互關係。
uniform
一致內容：此區段目前的內容形成一個邏輯單位，且會依序處理。 [預設值]
sample sample¶指出此區段是否為來源文件的樣本，若為樣本，則說明出自哪一部分。
狀態 非必備的
資料類型 teidata.enumerated
合法的值是:
initial
區段的末端無資料
medial
區段的起始與末端無資料
final
區段的起始部分無資料
unknown
來源文件中的樣本資料位置不明
complete
區段非樣本 [預設值]
