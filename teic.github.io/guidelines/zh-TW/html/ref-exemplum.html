<exemplum>

<exemplum> 包含說明某一元素使用的單一範例，可任意配合註解段落。 [22.5 Element Specifications]
組件tagdocs — Documentation Elements
屬性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.typed (@type, @subtype) att.translatable (@versionDate)

tagdocs: attDef classSpec dataSpec elementSpec macroSpec moduleSpec

例子
<exemplum>
 <p>
  <gi>名稱</gi>元素可以同時用於人名以及地名：</p>
 <eg xml:space="preserve"><![CDATA[ <q>我親愛的 <name           type="person">班耐特</name>先生，</q> 有一天太太對他說，           <q>你聽到<name type="place">尼得斐莊園</name>           終於租出去了嗎？</q>]]></eg>
 <p>如上所示，<att>屬性值</att> 可用來分辨人名或地名。</p>
</exemplum>
