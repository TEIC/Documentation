<attDef>

<attDef> (屬性定義) 包含單一屬性的定義。 [22.5.3 Attribute List Specification]
組件tagdocs — Documentation Elements
屬性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.identified (@ident, @predeclare, @module) (att.combinable (@mode) (att.deprecated (@validUntil)) )
usage usage¶說明屬性或元素的必備性。
狀態 非必備的
資料類型 teidata.enumerated
合法的值是:
req
(必備的)
rec
(推薦的)
opt
(非必備的) [預設值]
ns ns¶(名稱空間) 指明該屬性所屬的名稱空間。
狀態 非必備的
資料類型 teidata.namespace
預設值 http://www.tei-c.org/ns/1.0
可包含在於
tagdocs: attList
可包含
core: desc gloss
tagdocs: altIdent constraintSpec datatype defaultVal equiv exemplum remarks valDesc valList
例子
<attDef usage="rec" ident="type">
 <desc>具體指定慣例上使用的分部名稱，例如：
 <val>幕</val>, <val>卷</val>, <val>冊</val>, <val>節</val>, <val>篇章</val>,
   等。</desc>
 <datatype>
  <dataRef key="teidata.word"/>
 </datatype>
</attDef>

element attDef
{
   att.global.attributes,
   att.global.rendition.attributes,
   att.global.linking.attributes,
   att.global.analytic.attributes,
   att.global.facs.attributes,
   att.global.change.attributes,
   att.global.responsibility.attributes,
   att.global.source.attributes,
   att.identified.attributes,
   att.combinable.attributes,
   att.deprecated.attributes,
   attribute usage { "req" | "rec" | "opt" }?,
   attribute ns { teidata.namespace }?,
   (
      ( model.glossLike | model.descLike )*,
      datatype?,
      constraintSpec*,
      defaultVal?,
      ( valList | valDesc+ )?,
      exemplum*,
      remarks*
   )
}
