att.internetMedia

att.internetMedia provides attributes for specifying the type of a computer resource using a standard taxonomy.
組件tei — The TEI Infrastructure
成員att.media [binaryObject graphic] equiv ptr ref
屬性屬性
mimeType mimeType¶ (MIME media type) MIME協定種類
狀態 非必備的
資料類型 1–∞ 次可出現: teidata.word 以空白鍵隔開
例子

In this example mimeType is used to indicate that the URL points to a TEI XML file encoded in UTF-8.

<ref mimeType="application/tei+xml; charset=UTF-8"
