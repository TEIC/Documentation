<supportDesc>

<supportDesc> 匯集描述手稿書寫部分的物質載體元素。 [10.7.1 Object Description]
組件msdescription — Manuscript Description
屬性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
material material¶載體主要組成材料的簡短用途定義名稱
狀態 非必備的
資料類型 teidata.enumerated
被推薦的值包含:
paper
parch
(parchment)
mixed
可包含在於
msdescription: objectDesc
可包含
core: p
header: extent
linking: ab
msdescription: collation condition foliation support
例子
<supportDesc>
 <support> Parchment roll with <material>silk</material> ribbons.
 </support>
</supportDesc>

   att.global.attributes,
   att.global.rendition.attributes,
