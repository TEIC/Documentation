註

A view is a particular form of stage direction.

例子
<view>
 <name>馬克思</name>隨他女兒到窗邊。<hi>雨水</hi> 打在他臉上--
</view>
<view>
 <camera>馬克思的視點</camera> 他看見一些打開的窗戶，而就在他公寓對面，有一個<hi>男子</hi> 正打開富麗堂皇的大門。--
</view>
