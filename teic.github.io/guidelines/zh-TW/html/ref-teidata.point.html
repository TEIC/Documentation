teidata.point

teidata.point defines the data type used to express a point in cartesian space.

SchematronSeveral standards bodies, including NIST in the USA, strongly recommend against ending the representation of a number with a decimal point. So instead of ‘3.’ use either ‘3’ or ‘3.0’.
