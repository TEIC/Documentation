<constraint>

<constraint> (constraint rules) the formal rules of a constraint [22.5 Element Specifications]
組件tagdocs — Documentation Elements
屬性att.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source))
可包含在於
tagdocs: constraintSpec
可包含僅文字符號資料
例子
<constraintSpec ident="isoconstraint"
 scheme="schematron">
 <constraint>
  <s:assert test="tei:fileDesc/tei:titleStmt/tei:title[@type='main']"> a main title must be supplied
  </s:assert>
 </constraint>
</constraintSpec>

<constraintSpec ident="local"
 scheme="private">

 </constraint>
</constraintSpec>
