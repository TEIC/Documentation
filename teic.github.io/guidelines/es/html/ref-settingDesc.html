<settingDesc>

<settingDesc> (descripción de realización) describe en prosa, o mediante una serie de elementos de realización, la realización o realizaciones dentro de las cuales una interacción lingüística tiene lugar. [15.2 Contextual Information 2.4 The Profile Description]
Módulocorpus — Language Corpora
Atributosatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.declarable (@default)
Miembro de
model.profileDescPart
Contenido en
header: profileDesc
Puede contener
core: p
corpus: setting
linking: ab
namesdates: listPlace place
Nota

May contain a prose description organized as paragraphs, or a series of setting elements. If used to record not settings of language interactions, but other places mentioned in the text, then place optionally grouped by listPlace should be preferred.

Ejemplo
<settingDesc>
 <p>Texts recorded in the
   Canadian Parliament building in Ottawa, between April and November 1988 </p>
</settingDesc>
