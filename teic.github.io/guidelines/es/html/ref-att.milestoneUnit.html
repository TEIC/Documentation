att.milestoneUnit

att.milestoneUnit provides an attribute to indicate the type of section which is changing at a specific milestone. [3.10.3 Milestone Elements 2.3.6.3 Milestone Method 2.3.6 The Reference System Declaration]
Módulocore — Elements Available in All TEI Documents
Miembrosmilestone refState
AtributosAtributos
unit unit¶proporciona un nombre convencional para el tipo de sección que cambia en este hito.
Estado Necesario
Tipo de datos teidata.enumerated
Los valores sugeridos incluyen:
page
cambio físico de página (sinónimo del elemento <Pb>).
column
cambio de columna.
line
salto de línea (sinónimo del elemento <libra>).
book
cualquier unidad llamada libro, pliego, etc.
poem
poemas individuales de una colección.
canto
cantos u otras secciones importantes de un poema.
speaker
changes of speaker or narrator.
stanza
estrofas dentro de un poema, de un libro, o de un canto.
act
actos dentro de un drama.
scene
escenas dentro de un drama o de un acto.
section
secciones de cualquier tipo.
absent
pasajes no presentes en la edición de referencia.
unnumbered
passages present in the text, but not to be included as part of the reference.
