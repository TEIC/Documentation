att.measurement

att.measurement proporciona atributos que representen una medición regularizada o normalizada.

Nota

Si la medida que se representa no se expresa con una unidad determinada, pero constituye un número de elementos discretos, se debe utilizar la unidad cuenta, o el atributo unidad se puede dejar sin especificar.

Dondequiera que sea apropiado, un nombre de la unidad SI reconocido debe ser utilizado (véase http://www.bipm.org/en/publications/si-brochure/; http://physics.nist.gov/cuu/Units/). La lista mencionada serà más indicativa que exhaustiva.

Nota

Generalmente cuando el material se compone de entidades discretas, se debe usar la forma plural, incluso cuando la medida se refiere solamente a una de ellas.

Nota

Esta clase de atributo proporciona un trío de atributos que se pueden utilizar para regularizar los valores de medida que se codifican, o normalizarlos en un sistema de medida estándar.

Nota

La unidad debe ser nombrada normalmente usando la abreviatura estándar para una unidad SI (véase http://www.bipm.org/en/publications/si-brochure/; http://physics.nist.gov/cuu/Units/). Sin embargo, los codificadores pueden también especificar las medidas usando unidades definidas informalmente, por ejemplo como líneas o caracteres.
