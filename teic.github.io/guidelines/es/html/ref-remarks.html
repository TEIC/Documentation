<remarks>

<remarks> contiene cualquier comentario o explicación relativa al uso de elementos, atributos, clases o entitades no documentados de otra manera en el elemento que los contiene. [22.5 Element Specifications 22.5.3 Attribute List Specification 22.6 Class Specifications 22.7 Macro Specifications]
