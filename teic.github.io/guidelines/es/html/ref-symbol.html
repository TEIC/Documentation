<symbol>

<symbol> (valor simbólico) representa la parte del valor de una especificación de valor de rasgo que contiene uno de una lista finita de símbolos. [18.3 Other Atomic Feature Values]
Móduloiso-fs — Feature Structures
Atributosatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.datcat (@datcat, @valueDatcat)
value value¶proporciona el valor simbólico para un rasgo, uno de una lista finita que puede ser especificada en una declaración de rasgo.
