att.citing

att.citing provides attributes for specifying the specific part of a bibliographic item being cited. [1.3.1 Attribute Classes]
Módulotei — The TEI Infrastructure
MiembrosbiblScope citedRange
AtributosAtributos
unit unit¶identifies the unit of information conveyed by the element, e.g. columns, pages, volume, entry.
Estado Opcional
Tipo de datos teidata.enumerated
Los valores sugeridos incluyen:
volume
(el elemento contiene el número de un volumen.) el elemento contiene el número de volumen.
issue
el elemento contiene el número de la edición, o los números del volumen y de la edición.
page
(el elemento contiene un número de página o el intervalo de páginas.) el elemento contiene una página o un intervalo de páginas.
line
the element contains a line number or line range.
chapter
(capítulo) el elemento contiene la indicación del capítulo (número y/o el título)
part
el elemento identifica una parte de un libro o de una colección.
column
the element identifies a column.
entry
the element identifies an entry number or label in a list of entries.
from from¶specifies the starting point of the range of units indicated by the unit attribute.
Estado Opcional
Tipo de datos teidata.word
to to¶specifies the end-point of the range of units indicated by the unit attribute.
Estado Opcional
Tipo de datos teidata.word
