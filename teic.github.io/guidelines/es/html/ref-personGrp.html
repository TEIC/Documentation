<personGrp>

<personGrp> (grupo de personas) describe a un grupo de individuos considerados como una única persona para fines analíticos. [15.2.2 The Participant Description]
Módulonamesdates — Names, Dates, People, and Places
Atributosatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) (att.global.source (@source)) att.sortable (@sortKey)
role role¶especifica el role de ese grupo de participantes en la interacción.
Estado Opcional
Tipo de datos teidata.enumerated
sex sex¶especifica el sexo del grupo de participantes.
Estado Opcional
Tipo de datos 1–∞ apariciones de teidata.sex separado por espacio en blanco
age age¶especifica el intervalo de edad del grupo de partecipantes.
Estado Opcional
Tipo de datos teidata.enumerated
size size¶especifica la dimensión o dimensiones aproximadas del grupo.
Estado Opcional
Tipo de datos 1–∞ apariciones de teidata.word separado por espacio en blanco

Nota

May contain a prose description organized as paragraphs, or any sequence of demographic elements in any combination.

The global xml:id attribute should be used to identify each speaking participant in a spoken text if the who attribute is specified on individual utterances.

Ejemplo
<personGrp xml:id="pg1" role="audience"
 sex="mixed" size="approx 50"/>
