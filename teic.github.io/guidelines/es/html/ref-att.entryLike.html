att.entryLike

att.entryLike agrupa las diferentes modalidades de entradas de diccionarios [9.1 Dictionary Body and Overall Structure 9.2 The Structure of Dictionary Entries]

AtributosAtributosatt.typed (type, @subtype)
type type¶indica el tipo de entrada en diccionarios que contienen de diferentes tipos.
Estado Opcional
Tipo de datos teidata.enumerated
Los valores sugeridos incluyen:
main
una entrada principal (valor por defecto). [Por defecto]
hom
(un homógrafo con una entrada separada) agrupa la información referente a un homógrafo dentro de una entrada.
xref
(una breve entrada cuya única función es apuntar a otra entrada principal (p.ej. para formas de un verbo irregular o para otras variantes ortográficas: was señala a be, o esthete a aesthete).) una entrada reducida cuya única función es la de señalar otra entrada principal (p.ej. para las formas de un verbo irregular o para los deletreos variables: era señala a es, o esthete a esteta).
affix
una entrada para un prefijo, un infijo, o un sufijo.
abbr
(una entrada para una abreviatura) una entrada para una abreviatura.
supplemental
una entrada suplementaria (para el uso en diccionarios que presentan una edición complementaria al trabajo principal, en el cual la cual se incluye la información actualizada sobre las entradas).
foreign
una entrada para una palabra extranjera en un diccionario monolingüe.
