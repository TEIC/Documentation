Formal Specification for ODD and ODD Processing THIS IS A DRAFT IN DEVELOPMENT. Although the TEI Guidelines includes a chapter on using ODD (Documentation Elements), and the elements and attributes for ODD are defined in the P5 schema specification, there is no formal documentation to guide developers or maintainers of TEI ODD processors. That is the purpose of this document. It is being developed as part of the ATOP (Another TEI ODD Processor) teamʼs work to create a replacement for the existing TEI Stylesheets modules which generate RELAXNG and Schematron schemas from TEI ODD files.
